Mục tiêu nghề nghiệp

Xin chào! Tôi là Bùi Nhật Huy, sinh viên chuyên ngành Kỹ thuật phần mềm với định hướng phát triển trong lĩnh vực Backend Development. Tôi đặc biệt quan tâm đến kiến trúc hệ thống, tối ưu cơ sở dữ liệu, bảo mật ứng dụng và xây dựng API theo chuẩn RESTful. Tôi có kinh nghiệm làm việc với các công nghệ Backend như Java/Spring Boot, ASP.NET Core và Node.js, cùng với các hệ quản trị cơ sở dữ liệu như PostgreSQL, MySQL và SQL Server. Tôi luôn chú trọng viết code rõ ràng, có cấu trúc tốt, tuân thủ nguyên tắc Clean Code và đảm bảo tính bảo mật tối đa cho hệ thống.

API Development

Mini Social Media API

Hệ thống API mạng xã hội thu nhỏ cho phép người dùng đăng tải nội dung, kết nối và tương tác với nhau. Dự án được xây dựng với kiến trúc RESTful chuẩn, chú trọng vào tính bảo mật và khả năng mở rộng.

  • Hệ thống Xác thực & Phân quyền bằng JWT và Role-based Access Control (RBAC).
  • Quản lý bài viết linh hoạt, hỗ trợ đăng tải hình ảnh qua MultipartFile.
  • Tính năng tương tác: Like/Unlike bài viết và hệ thống bình luận (CRUD) đầy đủ.
  • Hệ thống Follow: Cho phép người dùng theo dõi và xây dựng mạng lưới kết nối.
  • Tích hợp Swagger/OpenAPI 3.0 giúp tự động hóa tài liệu và kiểm thử API.
