Mục tiêu nghề nghiệp

Xin chào! Tôi là Bùi Nhật Huy, sinh viên chuyên ngành Kỹ thuật phần mềm với định hướng phát triển trong lĩnh vực Backend Development. Tôi đặc biệt quan tâm đến kiến trúc hệ thống, tối ưu cơ sở dữ liệu, bảo mật ứng dụng và xây dựng API theo chuẩn RESTful. Tôi có kinh nghiệm làm việc với các công nghệ Backend như Java/Spring Boot, ASP.NET Core và Node.js, cùng với các hệ quản trị cơ sở dữ liệu như PostgreSQL, MySQL và SQL Server. Tôi luôn chú trọng viết code rõ ràng, có cấu trúc tốt, tuân thủ nguyên tắc Clean Code và đảm bảo tính bảo mật tối đa cho hệ thống.
